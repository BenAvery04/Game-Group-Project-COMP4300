## About

This is the project repo for COMP4300 F25 term group project. The project was created using a game engine developed throughout the course in assignments using C++. The game, titled "One Man Army", features 3 levels, a boss fight, multiple power-ups and weapons, various enemies with AI, a built-in level editor using the ImGui library and various lighting and shader effects.

 ## How to run
The game can be run with `COMP4300_Project.exe` in the Game folder. All other assets in the Game folder must be included
